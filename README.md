# TP Optique : Spectroscopie à prisme

Ce dépôt contient les ressources et le compte rendu du Travail Pratique n°3 portant sur l'étude d'un prisme à l'aide d'un goniomètre.

## 🎯 Objectifs du TP
L'objectif principal est de caractériser un prisme en verre en utilisant des techniques de précision :
* Apprendre le maniement et les réglages d'un goniomètre de précision.
* Mesurer l'angle utile A du prisme par réflexion.
* Relever la courbe de déviation D = f(alpha_i).
* Déterminer l'indice de réfraction n du verre au minimum de déviation.

## 🛠️ Dispositif Expérimental
Le montage s'appuie sur les éléments suivants :
* **Goniomètre** : composé d'un collimateur pour obtenir un faisceau parallèle, d'une plateforme mobile et d'une lunette de visée autocollimatrice. La lecture des angles est réalisée via un microscope muni d'un vernier précis à la minute d'arc.
* **Source lumineuse** : lampe spectrale au sodium émettant une raie jaune de longueur d'onde lambda = 589,3 nm.
* **Prisme** : milieu dispersif placé sur la plateforme dont on cherche à mesurer les caractéristiques.
